## Project Summary
Nature Inc is a professional landing page for a non-profit organization focused on system design for nature restoration and community empowerment. The organization develops replicable legal-tech-financial structures for nature-based projects worldwide.

## Tech Stack
- Framework: Next.js 15+ (App Router)
- Styling: Tailwind CSS 4 (via @tailwindcss/postcss)
- Fonts: Playfair Display (Serif), Lato (Sans-serif)
- Icons: Lucide React
- Components: Shadcn UI (Radix UI)
- Animation: Framer Motion / Motion

## Architecture
- `src/app`: Application routes and layout
- `src/components`: Reusable UI components
- `public`: Static assets (logo, team photos)

## User Preferences
- Clean, editorial, trustworthy aesthetic.
- Organic/natural feel with generous whitespace.
- Subtle animations on scroll and hover.
- Brand colors: Brown Dark (#4A3728), Brown Medium (#5D4632), Green Primary (#5B8C51), Green Light (#7AAD6D), Cream (#FAF8F5).

## Project Guidelines
- Follow the provided design brief for section structure and content.
- Use the most modern Next.js and Tailwind patterns.
- Ensure full responsiveness and accessibility.
- Maintain a cohesive aesthetic using CSS variables defined in globals.css.
